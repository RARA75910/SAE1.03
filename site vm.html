Configuration VirtualBox

Pourquoi utiliser une machine virtuelle ?

Une machine virtuelle permet de créer un environnement isolé et sécurisé pour le développement. Elle offre plusieurs avantages :

  • Isolation complète de l'environnement de développement
  • Possibilité de tester différentes configurations
  • Protection du système hôte
  • Facilité de sauvegarde et de restauration

Prérequis

  • Visual C++ 2019 Redistributable
  • Virtualisation BIOS/UEFI activée
  • 8 GB RAM minimum
  • 20 GB espace disque

Installation

  1. Télécharger VirtualBox
  2. Installer les prérequis
  3. Exécuter l'installation
  4. Redémarrer le système

Gestion des Utilisateurs

Création des Comptes

# Création administrateur
sudo useradd -m -G sudo admin_GrXX_BinomeYY
sudo passwd admin_GrXX_BinomeYY

# Création utilisateurs
sudo useradd -m BinomeYY_1
sudo useradd -m BinomeYY_2

Configuration des Groupes

# Création groupe
sudo groupadd BinomeYY
sudo usermod -a -G BinomeYY BinomeYY_1
sudo usermod -a -G BinomeYY BinomeYY_2

Configuration Apache

Installation

sudo apt update
sudo apt install apache2
sudo systemctl start apache2
sudo systemctl enable apache2

Configuration

# Configuration hôte virtuel
sudo nano /etc/apache2/sites-available/000-default.conf

# Activation du site
sudo a2ensite 000-default.conf
sudo systemctl reload apache2
